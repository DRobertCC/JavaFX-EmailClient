# Email Client with JavaFX

## Description

In this desktop app people can view and create emails, configure themes and the overall look and feel of the app.

## Features and Technologies included
-  Login
-  Email actions
-  Themes: Stages & CSS
-  JavaMail
-  Multithreading (Non blocking UI)
